
# Comandos Básicos do Git

## O que são comandos do Git?

Os comandos do Git são utilizados para controlar as versões de um projeto. Eles permitem criar repositórios, acompanhar alterações, salvar mudanças e colaborar com outras pessoas.

## Principais comandos

### git --version

Verifica se o Git está instalado e mostra a versão instalada.

### git init

Cria um novo repositório Git.

### git status

Mostra o estado atual do repositório, indicando arquivos modificados, adicionados ou não rastreados.

### git add

Adiciona arquivos para a área de preparação (staging).

Exemplo:

```bash
git add .
```

### git commit

Salva as alterações no histórico do projeto.

Exemplo:

```bash
git commit -m "Primeiro commit"
```

## Conclusão

Esses são alguns dos comandos fundamentais do Git. Com eles, é possível iniciar um projeto, acompanhar mudanças e registrar versões do código de forma organizada.
```bash
git add .
```
