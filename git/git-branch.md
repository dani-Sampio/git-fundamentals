#Git branch 

## O que é uma branch?
uma branch é uma ramificação do projeto. Ela  permite desenvolver novas funcionalidade ou corrigir problemas sem altere a versão principal do código.
A branch principal de um repositório geralmente é chamada de 'main'.

## Por que usar branch?
As branches permitem que diferentes pessoas trabelhem no mesmo projeto ao mesmo tempo, sem interferir uma no trabalho das outras.
Também são úteis para testar novas ideias antes de adicioná-las ao projeto principal.

## Sintaxe
Criar uma nova branch:
'''bash
git branch nome-da-branch
'''

Liatar todas as branches:

'''bash
git branch
'''

## Exemplo
Criar uma branch chamada 'nova-funcionalidade':

'''bash
git branch nova-funcionalidade
'''

## Vantagens
- Permite trabalhar em novas funcionalidade com segurança.
- Evita alterações diretas na branch principal.
- Facilita o trabalho em equipe.
- Organiza melhor o desenvolvimento do  projeto.
- 
## Resumo
O comando 'git branch' é utiloizado para criar e listar branches, permintindo que diferentes versões do projeto sejam desenvolvidas de forma independente.
