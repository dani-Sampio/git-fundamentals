# Git Ignore

## O que é o arquivo .gitignore?

O .gitignore é um arquivo utilizado para informar ao Git quais arquivos e pastas não devem ser monitorados (rastreados) ou enviados para o repositório.

Ele é muito útil para evitar o envio de arquivos temporários, configurações pessoais e informações sensíveis.

## Exemplo

text
node_modules/
.env
.vscode/
*.log


## Quando utilizar?

Utilize o .gitignore sempre que houver arquivos que não precisam fazer parte do projeto ou que contenham informações confidenciais.

## Vantagens

- Evita enviar arquivos desnecessários.
- Protege informações sensíveis.
- Mantém o repositório mais organizado.
- Facilita o trabalho em equipe.

## Resumo

O .gitignore é um arquivo que define quais arquivos e pastas o Git deve ignorar durante o controle de versão.
