# O que é Git ?

## Definição
Git é um sistema de controle de versão distribuído, criado por Linus Torvalds em 2005.
Ele permite acompanhar alterações em arquivos,trabalho em equipe e recuperar versões anteriores de um projeto.

## Principais vantagens
- Controlar versões de projeto.
- Trabalhar em equipe sem sobrescrever arquivos.
- Recuperar versões anteriores.
- Organizar o histórico de alterações.

## Onde o Git é urilizado?
O Git é utilizado por desenvolvedores de software, profissionais de QA, analistas de dados e equipes de tecnologia para gerenciar projetos de forma organizada.

## Exemplo de uso
Imagine que você alterou um arquivo por engano.
Com o Git, é possivel recuperar versões anteriores e visualizar todas as mudanças realizadas no projeto.
